# 爆款标题公式

B站标题上限较宽但**推荐控制在20字以内**，抖音更短更好。每次只产出 **3 个精华标题**——宁可3个都能打，也不要5个里凑数。

## 一个好标题的本质：三样东西的碰撞

标题不是"概括视频讲了啥"，而是**让人忍不住点进去**。真正能打的标题，是这三样的碰撞，缺一不可：

1. **具体锚点**——一个真实的数字、名字或事件（"套在1800""跌70%""财报当天"）。它证明你不是标题党，给可信度，也让特定人群对号入座。
2. **情绪钩子**——只戳**一根**神经：恐惧（怕亏/怕踏空）、贪婪（想抄底）、好奇（到底为什么）、认同（说出我想说的）。一个标题贪多戳好几根，反而一根都戳不进去。
3. **信息缺口**——藏起一个答案，只有点进去才能填。缺口的火候是关键（见下）。

一句话：**具体锚点给可信、情绪钩子给冲动、信息缺口给理由。** 三者齐了，标题就立住了。

## 信息缺口的火候（最容易做错的一步）

缺口太满或太空都不点。校准它：

- **太满（差评）**：结论全抖在标题里，点进去没有新东西。"茅台被高估了，建议卖出"——都说完了，何必点？
- **太空（差评）**：勾不出具体在讲什么，观众不敢赌那几分钟。"这只票有个大秘密"——什么票？什么秘密？不可信。
- **刚好（好评）**：给出**足够具体的悬念**，让人觉得"答案就在里面、而且我需要它"。"套在茅台1800的人，现在只有一条路"——知道讲什么、戳中我、但不点不知道那条路是什么。

## 六个可用角度（每次选差异最大的3个）

角度是工具箱，不是填空模板——选完角度，回到上面"三样碰撞"去打磨每一个字。

### 1. 悬念留白型
藏起一个必须点进去才能填的信息缺口。
- "跌了30%的茅台，我反而加仓了，就因为这一个数字。"
- "宁德时代这份财报，有一行字比营收更重要。"

### 2. 数字冲击型
用**反差大、够具体**的数字抓眼球——数字要让人意外，不只是"有个数字"。
- "3年跌70%，宁德时代的底还没到？"
- "账上800亿现金，股价却腰斩，问题出在哪。"

### 3. 争议观点型
给一个反共识的判断，激发认同或反驳欲。**站队要狠，别骑墙。**
- "所有人都在割肉，我却把这只票加到满仓。"
- "英伟达的故事，我认为讲不下去了。"

### 4. 提问代入型
问出观众心里正纠结的问题，让特定人群觉得"说的就是我"。
- "套在茅台1800的人，现在该割还是该扛？"
- "现在上车比亚迪，是抄底还是接盘？"

### 5. 结论前置型
把结论/建议直接放标题——但**留一半**，别把理由也抖光。
- "一句话：茅台能拿，但别追，原因就一个。"
- "看完这条，你就知道这票该不该卖。"

### 6. 蹭热点/事件型
绑定最新事件吃平台热点流量，事件要"新鲜到今天"。
- "涨停背后：主力今天到底干了什么。"
- "财报暴雷当天，我把这只票的账重算了一遍。"

## 打磨清单（写完每个标题逐条自查）

- **前6个字够不够狠**？手机信息流会截断标题，最勾人的词（数字、反差、身份）要**放最前面**，别浪费在"关于""浅谈"上。
- **只戳了一根神经吗**？一个标题一种情绪，别又恐惧又好奇又贪婪。
- **数字够不够意外**？平淡的数字不如不放。要么反差大，要么反常识。
- **动词带不带电**？用"割肉""踏空""闷杀""抄底"这种有画面、有情绪的词，别用"分析""探讨""了解"。
- **缺口火候对吗**？对照上面"太满/太空/刚好"，确认是"刚好"。
- **删得动吗**？能删的字都删掉，20字以内越短越冲，抖音再压更短。

## 输出格式

产出**3个**候选，覆盖3种不同角度，格式如下：

```
1. [标题]（角度：悬念型）⭐⭐⭐⭐⭐ 推荐
2. [标题]（角度：争议型）⭐⭐⭐⭐
3. [标题]（角度：提问型）⭐⭐⭐⭐
```

推荐指数最高的放第一个，并用一句话说清它为什么最可能爆——点明它踩中了哪根情绪神经、缺口卡在哪里。

## 红线（不可越）

- 标题里的数字、事实必须和口播稿、真实数据一致，不能为了点击率编造。
- 标题承诺的东西**必须在视频里兑现**，且别太早兑现——比如标题说"3个关键价位"，正文就该把价位放到中后段，否则开头抖完包袱，后面没人看，完播率崩。
- 不用"必涨""稳赚""内幕""史诗级"这类违规、夸张或误导词。
- 可以有情绪，但不做绝对承诺——"可能""或许""我认为"是安全垫。
