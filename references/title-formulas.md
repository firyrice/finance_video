# 爆款标题公式

B站标题上限较宽但**推荐控制在20字以内**，抖音更短更好。好标题的本质：在不骗人的前提下，制造信息缺口或情绪张力，让人忍不住点进来。

## 六大公式（每次从中选不同风格凑够5个）

### 1. 悬念留白型
留一个必须点进去才能填补的信息缺口。
- "跌了30%的茅台，我为什么反而加仓了？"
- "宁德时代这份财报，藏着一个没人注意的信号"

### 2. 数字冲击型
用具体、反差大的数字抓眼球。
- "3年跌70%，宁德时代还有救吗？"
- "市盈率只有8倍，这只银行股被低估了？"

### 3. 争议观点型
给一个反共识或站队的判断，激发认同或反驳欲。
- "别再吹英伟达了，我说说它的三个风险"
- "所有人都在割肉，我却觉得机会来了"

### 4. 提问代入型
问出观众心里正纠结的问题。
- "套在茅台1800的人，现在该怎么办？"
- "现在上车比亚迪，是抄底还是接盘？"

### 5. 结论前置型
直接把结论/建议放标题，吸引想要答案的人。
- "一句话结论：茅台现在能拿不能追"
- "看完这条，你就知道该不该卖了"

### 6. 蹭热点/事件型
绑定最新事件，吃平台热点流量。
- "涨停背后：主力资金今天干了什么"
- "财报暴雷当天，我把这只票的账算给你看"

## 输出格式

产出5个候选，覆盖至少3种不同风格，格式如下：

```
1. [标题]（风格：悬念型）⭐⭐⭐⭐⭐ 推荐
2. [标题]（风格：数字型）⭐⭐⭐⭐
3. [标题]（风格：争议型）⭐⭐⭐⭐
4. [标题]（风格：提问型）⭐⭐⭐
5. [标题]（风格：结论型）⭐⭐⭐
```

推荐指数最高的放第一个，并可以用一句话说明为什么它最可能爆。

## 注意

- 标题里的数字、事实必须和口播稿、真实数据一致，不能为了点击率编造。
- 不用"必涨""稳赚""内幕"这类违规或误导词。
- 可以有情绪，但不做绝对承诺——"可能""或许""我认为"是安全垫。
