# 口播稿结构模板

目标：一条 7-10 分钟（约2500-3500字）、能对着念、观点鲜明又通俗的财经口播稿。观众是想搞懂"这票到底能不能买"的普通人，但也有一部分懂行的，所以既要通俗，又要有干货，不能水。

## 整体结构

按下面六段来写。每段标注了大致时长和目的，字数按 7-10 分钟分配。

### 1. 黄金开场（0-15秒 / 约150字）
前15秒决定完播率。**不要**说"大家好我是XX""今天我们来聊聊"这种废话。三种高效开场，任选其一：

- **抛结论**："先说结论：现在的XX，我不建议追。为什么？看完这三点你就明白了。"
- **抛悬念/数字**："一个月跌了30%的XX，昨天突然涨停。是主力回来了，还是最后的诱多？"
- **抛冲突**："所有人都说XX要完了，但我看完最新财报，反而想加仓。"

### 2. 背景铺垫（约300-400字）
快速交代"发生了什么"。用最新的事件/数据把观众拉到同一个信息水位：股价现在多少、最近发生了什么催化、市场在吵什么。数字要具体，但一次别堆太多，讲一个消化一个。

### 3. 基本面分析（约700-900字）
硬核部分之一。讲清楚这家公司**靠什么赚钱、赚得怎么样、未来还能不能赚**：
- 主营业务和商业模式（一句话讲明白它怎么赚钱）
- 最新财报关键数据：营收/净利润及增速、毛利率、现金流
- 护城河或风险点：为什么别人抢不走它的生意 / 它正面临什么威胁
- 把数字翻译成人话，比如"毛利率91%，意思是卖100块的酒，成本只有9块"

### 4. 技术面分析（约500-600字）
另一块硬核。讲图形和资金：
- 当前所处的位置（历史高位/低位、关键均线上下）
- 关键支撑位和压力位（给具体价格）
- 量能与趋势：放量还是缩量、突破还是破位
- 提醒：技术面是概率不是保证，用"如果站上X，大概率看向Y；如果跌破Z，要警惕"这种条件句

### 5. 估值 + 明确建议（约500-600字）
这是观众最想听的，也是最能体现观点的部分。**不要含糊**：
- 估值判断：当前PE/PB处在历史什么分位、和同行比贵还是便宜、给一个合理估值区间的判断
- 明确态度：买入 / 卖出 / 观望，并说清理由
- 给操作参考：什么价位可以考虑、什么价位该止损、适合什么风险偏好的人
- 用"我个人的看法是…""如果是我，我会…"这样的表达，既鲜明又留有余地

### 6. 收尾 + 风险提示 + 互动（约200字）
- 一句话总结全片核心观点
- **风险提示（必须有）**："以上仅为个人分析，不构成投资建议。股市有风险，入市需谨慎，请独立决策。"
- 互动引导："你怎么看XX？是准备上车还是已经下车了？评论区聊聊。觉得有用点个关注，下期分析你们点的票。"

## 语气与写法要求

- **口语化**：写完自己念一遍，拗口的地方就是要改的地方。多用短句。
- **通俗但不low**：专业名词第一次出现就用一句大白话解释，之后可以直接用。
- **有节奏感**：适当插入"划重点""记住这个数字""注意了"来提示关键信息，帮观众抓重点。
- **观点鲜明**：敢下判断，但用"我认为""大概率""从数据看"来承接，不做绝对承诺。
- **段落过渡**：段与段之间用口语衔接，比如"说完了基本面，我们再看看技术面怎么走"。

## 输出格式

用带时间轴/段落标题的形式输出，方便录制时找位置，例如：

```
【开场 · 0:00】
（正文……）

【基本面 · 约1:30】
（正文……）
```

## 一个开场范例（供参考语感，不要照抄）

> "先说结论：如果你现在手里拿着贵州茅台，我建议你拿住别动；但如果你想现在追进去，我劝你再等等。为什么同一支票，拿着和买入是两个建议？因为价格。今天这条视频，我用最新的财报和技术面数据，跟你把茅台到底值多少钱这件事，算清楚。"
