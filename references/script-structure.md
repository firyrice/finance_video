# 口播稿结构模板

目标：一条 7-10 分钟（约2500-3500字）、能对着念、观点鲜明又通俗的财经口播稿，出自财经账号 **「蛋炒饭财经」**。观众是想搞懂"这票到底能不能买"的普通人，但也有一部分懂行的，所以既要通俗，又要有干货，不能水。

三条贯穿全文的硬要求：
- **数据最新且正确**：文中所有股价、涨跌幅、市值、PE/PB、财报数字，必须是写稿当日（或最近交易日）的真实数据，写稿前用 WebSearch 再核对一次。
- **只说名字不报代码**：正文提到股票只说名字（"贵州茅台""宁德时代"），不要念股票代码。
- **标点用中文全角**：正文一律使用中文常用标点，逗号用「，」不用英文「,」，句号「。」、问号「？」、感叹号「！」、冒号「：」、分号「；」、顿号「、」，引号用「""''」，括号用「（）」，省略号用「……」。仅数字、股票代码、英文单词/短语内部保留英文半角符号，其余中文语境一律全角。

## 整体结构

按下面六段来写。每段标注了大致时长和目的，字数按 7-10 分钟分配。

### 1. 黄金开场（0-15秒 / 约150字）
前15秒决定完播率，而**前5秒决定观众走不走**，要分两层设计：

**前5秒（生死线，决定去留）**：第一句必须一口气能说完、控制在15字以内，且必须含一个具体数字或一个反差判断，**禁止任何从句铺垫和自我介绍**。三种高效开头，任选其一：
- **抛结论**："先说结论，现在的XX，我不建议追。"
- **抛悬念/数字**："一个月跌30%的XX，昨天突然涨停。"
- **抛冲突**："所有人都说XX要完了，我却想加仓。"

**5-15秒（决定要不要看完）**：接着把钩子展开，并**埋一个价值锚点**——把观众最想要的东西预告在开头、兑现在结尾，这是拉完播最有效的手段。例如"这条视频，我会给你3个关键价位""看完这三点，你就知道该不该买"。价值锚点要具体、可兑现，且真正的答案放到中后段揭晓，吊着观众看下去。

### 2. 背景铺垫（约300-400字）
快速交代"发生了什么"。用最新的事件/数据把观众拉到同一个信息水位：股价现在多少、最近发生了什么催化、市场在吵什么。数字要具体，但一次别堆太多，讲一个消化一个。

> 下面的第3-5段是文稿的干货主体，对应七维分析框架（基本面→估值，中间穿插行业、资金面、催化与风险）。**不必七个维度平均用力**，围绕本条选题的角度，把最相关的2-3个维度讲透，其余带过，避免变成流水账。

### 3. 基本面 + 行业（约700-900字）
硬核部分之一。讲清楚这家公司**靠什么赚钱、赚得怎么样、未来还能不能赚**：
- 主营业务和商业模式（一句话讲明白它怎么赚钱）
- 最新财报关键数据：营收/净利润及增速、毛利率净利率、ROE、现金流、负债率
- 护城河：品牌/技术/规模/网络效应——为什么别人抢不走它的生意；或它正面临什么威胁
- 行业赛道：行业处在成长还是成熟/衰退期、政策是顺风还是逆风、公司在产业链里的地位和议价权
- 把数字翻译成人话，比如"毛利率91%，意思是卖100块的酒，成本只有9块"；"ROE 30%，意思是股东每投100块，一年能赚回30块"

### 4. 技术面 + 资金面（约500-600字）
另一块硬核。讲图形和资金，回答"什么时候买、什么价位买"：
- 当前所处的位置（历史高位/低位、关键均线上下、多头还是空头排列）
- 关键支撑位和压力位（给具体价格）
- 量能与趋势：放量还是缩量、突破还是破位；MACD/RSI 是否超买超卖（点到为止，别堆指标）
- 资金动向：主力/北向资金/机构近期是流入还是流出、龙虎榜或持仓有没有异动
- 提醒：技术面是概率不是保证，用"如果站上X，大概率看向Y；如果跌破Z，要警惕"这种条件句

### 5. 估值 + 催化风险 + 明确建议（约500-600字）
这是观众最想听的，也是最能体现观点的部分。**不要含糊**：
- 估值判断：当前PE/PB/PS处在历史什么分位、和同行比贵还是便宜、PEG 有没有透支成长，给一个合理估值区间
- 未来变量：短期有什么催化剂（财报、新品、政策、景气拐点）；有什么被忽视的风险（业绩、政策、竞争、泡沫、减持）
- 明确态度：买入 / 卖出 / 观望，并说清理由
- 给操作参考：什么价位可以考虑、什么价位该止损/止盈、适合长线价值还是短线博弈的人
- 用"我个人的看法是…""如果是我，我会…"这样的表达，既鲜明又留有余地

### 6. 收尾 + 风险提示 + 互动（约200字）
- 一句话总结全片核心观点
- **风险提示（必须有）**："以上仅为个人分析，不构成投资建议。股市有风险，入市需谨慎，请独立决策。"
- **互动引导（必须有）**：先抛一个让人想评论的问题，再引导三连和关注，最后用固定收尾。以「蛋炒饭财经」的口吻，例如：
  - "你怎么看这只票？是准备上车还是已经下车了？评论区告诉我。"
  - "觉得这期分析有用的，帮我点个赞、加个收藏，方便以后回看；想继续看这种深度拆解的，记得关注『蛋炒饭财经』，你们点的票，下期我接着安排。"
  - 用一句稳定的口头禅收尾，强化人设，比如："我是蛋炒饭，我们下期再见。"
- 具体措辞每期可以微调，但"抛问题 → 点赞收藏关注蛋炒饭财经 → 下期再见"这个结构每条都要有。

## 语气与写法要求

- **口语化**：写完自己念一遍，拗口的地方就是要改的地方。多用短句。
- **通俗但不low**：专业名词第一次出现就用一句大白话解释，之后可以直接用。
- **有节奏感**：适当插入"划重点""记住这个数字""注意了"来提示关键信息，帮观众抓重点。每30-45秒给一个"小高潮"（一个数据、一个判断、一个金句），别把干货全堆在前2分钟。
- **观点鲜明**：敢下判断，但用"我认为""大概率""从数据看"来承接，不做绝对承诺。
- **段落过渡用钩子，不用平叙**：段与段之间不要用"说完了基本面，我们再看看技术面"这种平叙式过渡——这是最容易掉人的写法。改用**预告式钩子**推着观众进下一段，比如"但真正的问题在这""先记住这个数字，后面算价位要用""第二点更关键，很多人都忽略了"。每次过渡都给观众一个"再看下一秒"的理由。
- **消灭平坦段（防跳出）**：中段连续讲财报、估值数据是跳出重灾区。规则：连续讲的专业名词不超过2个就要停下来用大白话解释或类比一次（"这相当于你每年白干三个月"），并适时插一句散户视角（"评论区很多人纠结的就是这一点"）拉回代入感。
- **标点全角**：写完通读一遍，把中文语境里误用的英文标点（`,` `.` `?` `!` `:` `;`）全部换成中文全角（`，` `。` `？` `！` `：` `；`），只保留数字、代码、英文短语内部的半角符号。

## 输出格式

用带时间轴/段落标题的形式输出，方便录制时找位置，例如：

```
【开场 · 0:00】
（正文……）

【基本面 · 约1:30】
（正文……）
```

## 一个开场范例（供参考语感，不要照抄）

> "先说结论：如果你现在手里拿着贵州茅台，我建议你拿住别动；但如果你想现在追进去，我劝你再等等。为什么同一支票，拿着和买入是两个建议？因为价格。今天这条视频，我用最新的财报和技术面数据，跟你把茅台到底值多少钱这件事，算清楚。"
